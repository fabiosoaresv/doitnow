### Do it now
This project is based in Checklist, I'm use technology Ruby on Rails to development.

### How to run
- Dependencies
> ruby (2.5.1p57)

- Clone the project
`$ git clone https://github.com/fabiosoaresv/doitnow.git`

- Start development server
`$ bundle install`
`$ rake db:create db:migrate`
`$ rails s`
